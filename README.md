# Проект 2. Трекер самолётов

## Описание проекта
Система для отслеживания самолетов в воздушном пространстве с использованием API OpenSky Network и Nominatim.

## О проекте

Airplane Tracker - это приложение для мониторинга воздушного движения, которое позволяет:
- Получать данные о самолетах в реальном времени через OpenSky API
- Отслеживать самолеты по географическим координатам стран
- Сохранять информацию о самолетах в JSON-файл
- Фильтровать и сортировать данные по различным критериям

## Возможности

### Поиск самолетов
- Поиск по названию страны
- Автоматическое определение bounding box через Nominatim API
- Предустановленные координаты для популярных стран

### Работа с данными
- Сохранение информации в JSON-файл
- Защита от дубликатов
- Фильтрация по стране регистрации
- Фильтрация по диапазону высот
- Сортировка по высоте полета
- Получение топ N самолетов

### Интерактивное меню
📋 Доступные команды:
- 1️⃣ Поиск самолетов по стране
- 2️⃣ Показать топ N самолетов по высоте
- 3️⃣ Фильтр по стране регистрации
- 4️⃣ Фильтр по диапазону высот
- 5️⃣ Показать все сохраненные самолеты
- 0️⃣ Выход

## Тестирование

Проект использует `pytest` для тестирования.

### Запуск тестов

```bash
# Запуск всех тестов
poetry run pytest

# Запуск с детальным выводом
poetry run pytest -v

# Запуск конкретного модуля
poetry run pytest tests/test_masks_pytest.py -v

# Запуск с покрытием кода
poetry run pytest --cov=src --cov-report=html
